# 히든페이지 (Hidden Page) 🕵️

나무위키 문서의 숨겨진 단어를 맞추는 한국어 단어 추측 게임입니다.

## 🎮 게임 방법

1. **명사**는 ⬛로 숨겨져 있습니다
2. 단어를 입력해서 숨겨진 단어를 찾으세요
3. 모든 단어를 찾으면 게임 완료!

### 피드백 시스템
- ❄️ **Cold** - 문서에 없는 단어
- 🔥 **Warm** - 1~5개 발견
- 🎆 **Hot** - 6개 이상 발견!

## 🎨 주요 기능

- **라이트/다크 모드** - 🌙/☀️ 버튼으로 전환
- **잉크 번짐 효과** - 단어 공개 시 애니메이션
- **통계 시스템** - 플레이 기록 저장
- **반응형 디자인** - 모바일/PC 최적화

## 🛠️ 기술 스택

- **Frontend**: Vite + React
- **Styling**: Vanilla CSS (Pretendard 폰트)
- **State**: React Hooks + LocalStorage

## 🚀 실행 방법

```bash
npm install
npm run dev
```
