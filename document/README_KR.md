# 히든페이지 (Hidden Page) 🕵️

나무위키 문서의 숨겨진 단어를 맞추는 한국어 단어 추측 게임입니다.

## 🎯 주요 기능

- **마스킹 시스템**: 명사는 ⬛로 숨겨지고, 조사/동사/부호는 공개됩니다
- **피드백 연출**: Cold(❄️)/Warm(🔥)/Hot(🎆) 시각적 피드백
- **통계 시스템**: 플레이 횟수, 연승 기록, 평균 시도 횟수
- **반응형 디자인**: 모바일/PC 최적화

## 🎮 플레이 방법

1. 게임 시작 시 나무위키 문서가 표시됩니다
2. 명사는 ⬛로 숨겨져 있습니다
3. 단어를 입력하여 맞춰보세요
4. 모든 단어를 찾으면 게임 완료!
